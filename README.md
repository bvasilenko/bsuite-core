# bsuite-core

Shared CLI plumbing library for the b-* command-line suite.

## Install

Add the package to a Cargo workspace:

```toml
[dependencies]
bsuite-core = "0.1.0"
```

## Use

Use the exported traits and value types to connect command-line binaries to
prompt resolution, update checks, transcript storage, manifest overlays, exit
codes, visibility evidence, and host adapter bindings.
